# School Planner

A desktop school planner application built with Python, Tkinter, and SQLite.

The goal of this project is to provide students with a simple way to manage assignments, schedules, and academic tasks in a customizable desktop application.

## Features

### Assignments

- Add assignments
- Track due dates
- Mark assignments as completed
- Store data using SQLite

### Schedule

- Add classes
- Store class meeting times
- View schedule information

### Settings

- Theme customization
- Light mode
- Dark mode
- Custom background colors

## Technologies Used

- Python
- Tkinter
- SQLite

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/school-planner.git
```

Navigate to the project folder:

```bash
cd school-planner
```

Run the application:

```bash
python main.py
```
